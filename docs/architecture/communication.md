## 1. Frontend вҶ’ Backend: HTTP + Streaming

### жЁҷжә– Request/Response

жүҖжңүеүҚз«ҜеҲ°еҫҢз«Ҝзҡ„йҖҡиЁҠеқҮйҖҸйҒҺ Nginx еҸҚеҗ‘д»ЈзҗҶиҮі Core MonolithпјҢдёҰдҪҝз”Ё **HTTP REST**гҖӮ

```
Browser вҶ’ https://domain.com/api/finance/transactions вҶ’ Nginx вҶ’ Core Monolith
```

**ж…ЈдҫӢ**:
- `GET` з”Ёж–ји®ҖеҸ–пјҢ`POST` з”Ёж–је»әз«ӢпјҢ`PUT` з”Ёж–је®Ңж•ҙжӣҙж–°пјҢ`PATCH` з”Ёж–јйғЁеҲҶжӣҙж–°пјҢ`DELETE` з”Ёж–јеҲӘйҷӨ
- Request/response body дҪҝз”Ё JSON ж јејҸпјҲйҚөеҗҚжҺЎз”Ё camelCase д»Ҙзӣёе®№ JSпјү
- еҲҶй ҒпјҡдҪҝз”Ё `?page=1&limit=20` жҗӯй…Қ `X-Total-Count` жЁҷй ӯ
- йҢҜиӘӨиҷ•зҗҶпјҡдҪҝз”Ё `{ "detail": "message" }` жҗӯй…ҚйҒ©з•¶зҡ„ HTTP зӢҖж…Ӣзўј

### дёІжөҒ (SSE)

йҮқе°Қеҹ·иЎҢжҷӮй–“ијғй•·зҡ„дҪңжҘӯжҲ– LLM еӣһжҮүпјҢдҪҝз”Ё **Server-Sent Events**:

```
Browser вҶ’ GET /api/chat/stream (Accept: text/event-stream) вҶ’ Nginx вҶ’ Core
         вҶҗ data: {"chunk": "Hello"}\n\n
         вҶҗ data: {"chunk": " world"}\n\n
         вҶҗ data: [DONE]\n\n
```

**дҪ•жҷӮдҪҝз”Ё SSE иҲҮ WebSocket:**

| жЁҷжә– | SSE | WebSocket |
|----------|-----|-----------|
| ж–№еҗ‘ | Server вҶ’ Client (е–®еҗ‘) | йӣҷеҗ‘ |
| дҪҝз”Ёе ҙжҷҜ | LLM дёІжөҒгҖҒйҖІеәҰжӣҙж–° | иҒҠеӨ©гҖҒеҚіжҷӮеҚ”дҪң |
| йҮҚж–°йҖЈжҺҘ | е…§е»әиҮӘеӢ•йҮҚйҖЈ | йңҖжүӢеӢ•еҜҰдҪң |
| йҖҸйҒҺд»ЈзҗҶ | еҸҜзӣҙжҺҘйҖҸйҒҺ Nginx/CDN | йңҖ `Upgrade` ж”ҜжҸҙ |
| иӨҮйӣңеәҰ | з°Ўе–® | ијғиӨҮйӣң |

**й җиЁӯйҒёж“ҮпјҡSSE** -- д»ҘијғдҪҺзҡ„иӨҮйӣңеәҰж»ҝи¶і 90% зҡ„дёІжөҒйңҖжұӮгҖӮ

### жӘ”жЎҲдёҠеӮі

```
Browser вҶ’ POST /api/storage/upload (multipart/form-data) вҶ’ Nginx вҶ’ Core вҶ’ Object Store
```

## 3. дәӢд»¶й©…еӢ•йҖҡиЁҠ (Core е…§йғЁ)

е–®й«”жһ¶ж§Ӣе…§зҡ„жЁЎзө„й–“йҖҡиЁҠдҪҝз”Ё **Event Bus**гҖӮ

е®Ңж•ҙиҰҸзҜ„и©іиҰӢ [Event-Driven Architecture](./event-driven.md)гҖӮ

**ж‘ҳиҰҒ**пјҡ
- зӢҖж…Ӣи®ҠжӣҙпјҲйқһеҗҢжӯҘпјү вҶ’ Event Bus
- иіҮж–ҷжҹҘи©ўпјҲеҗҢжӯҘпјү вҶ’ Service importпјҲйҖІзЁӢе…§пјү
- еӨ–йғЁжңҚеӢҷе‘јеҸ« вҶ’ HTTPпјҲhttpxпјү

## 4. Core вҶ’ Hot-Path жңҚеӢҷ: HTTP + Events

Core Monolith иҲҮ Realtime еҸҠ Media жңҚеӢҷзҡ„йҖҡиЁҠж–№ејҸеҰӮдёӢпјҡ

| ж–№еҗ‘ | жЁЎејҸ | зҜ„дҫӢ |
|-----------|---------|---------|
| Core вҶ’ Realtime | HTTP API | з”ўз”ҹ LiveKit room token |
| Core вҶ’ Media | HTTP API | и«ӢжұӮ STT йҖҗеӯ—зЁҝ |
| Realtime вҶ’ Core | Redis Events | жҲҝй–“жҲҗе“ЎеҠ е…Ҙ |
| Media вҶ’ Core | Redis Events | йҖҗеӯ—зЁҝиҪүжҸӣе®ҢжҲҗ |

```python
# Core е‘јеҸ« Media жңҚеӢҷ
import httpx

async def request_transcription(audio_url: str, user_id: str):
    async with httpx.AsyncClient() as client:
        resp = await client.post("http://localhost:8831/transcribe", json={
            "audio_url": audio_url,
            "user_id": user_id,
        })
        return resp.json()
```
